Reach makes library packaging very easy: simply push your code to GitHub or BitBucket and tag it to mark a version number.0

Packages on GitHub may also activate the reach-pkg topic by configuring it in their repo's "About" configuration to enjoy greater discoverability.1

The package imports reference section details Reach's remote module import syntax, and the compile command reference describes the --install-pkgs command-line argument which allows remote packages to be fetched during compilation.2
